# Sarhad (Warpath repo) — Build Instructions

These instructions build the current Android prototype in this repository.

## Quick build
```bash
./scripts/build_apk.sh
```

## Output
Default debug APK output is generated by Gradle under:
```bash
app/build/outputs/apk/debug/app-debug.apk
```

## Optional versioned artifact copy
```bash
cp app/build/outputs/apk/debug/app-debug.apk APK/Warpath_vX_X_X.apk
```

## Notes
- Repository/package naming still uses `Warpath` in places while documentation is being migrated to **Sarhad**.
- The project targets Android and uses Gradle + Kotlin.
- If local Android SDK configuration is missing, create `local.properties` with your SDK path, for example:
  ```bash
  echo "sdk.dir=/opt/android-sdk" > local.properties
  ```
