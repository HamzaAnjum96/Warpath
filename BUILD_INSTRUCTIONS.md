# Warpath — Build Instructions

These instructions build the current Android prototype in this repository.

## Quick build
```bash
./scripts/build_apk.sh
```

## Direct Gradle build
```bash
./gradlew assembleDebug
```

## Output
The debug APK is generated by Gradle at:
```bash
app/build/outputs/apk/debug/app-debug.apk
```

## Artifact handling
Release/versioned APK distribution is handled by the CI pipeline artifacts. Do not commit APK binaries to this repository.

## Notes
- The project targets Android and uses Kotlin + Gradle.
- If local Android SDK configuration is missing, create `local.properties` with your SDK path, for example:
  ```bash
  echo "sdk.dir=/opt/android-sdk" > local.properties
  ```
